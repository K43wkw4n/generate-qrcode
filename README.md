# QR Studio — Free QR Code Generator

QR Studio is a fast, free, client-side QR code generator that helps marketers, designers, and everyday users create high-resolution, printable QR codes with custom colors, shapes, and optional center logos.

- Generate QR codes for URLs, text, WiFi, vCards, and more
- Customize dot shape, eye frames, colors, and logo placement
- Download high-resolution PNG, JPG, or SVG for web or print
- 100% client-side — no uploads, no account required

## Why use QR Studio

- Fast: instant preview and export
- Flexible: multiple styling options and logo support
- High quality: export at high resolution for print
- Privacy-first: everything runs in your browser

## Quick Start

```bash
npm install
npm run dev
# Build for production
npm run build
```

## SEO & Deployment Notes

- `index.html` contains descriptive title, meta description, Open Graph, Twitter Card, and JSON-LD structured data to help search engines index the app.
- Add the site to Google Search Console and submit `public/sitemap.xml` after deployment.
- Use an accessible landing page and server-side rendering or prerendering for best SEO results on dynamic content.

## Deployment

Deploy to any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages, S3 + CloudFront).
